Your mission

It is very simple. You have to write a nickname (blank is possible too) and click the button enter. That's all!
After that, a web chat appears, you can chat with other participants in real time (Only if you want. Chat is only to have fun :). It's not a part of the experiment.).
It works via P2P network and the messages are not stored on any server, the communication is peer to peer between web browsers. So, your messages doesn't pass through a server on Internet. Cool! isn't it?

How it works?

When you press "Enter" button, your browser received a list of peers and carry out a handshake process known as Signaling process. This exchange of messages (sdp messages) passes through a server known as Signaling Server. When the process is completed, your browser is able to comunicate with other browsers directly with a peer to peer conection. During the signaling process we (my team and me) are receiving the same information that you can see in the developer console (press F12 to see it). It's only time information. In any case, this is open source and you can check it in depth!
Note:This version only works properly with Chromium and Google Chrome (including Android version).
